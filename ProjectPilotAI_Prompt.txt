Build a production-grade, enterprise-level AI-Powered Project Management Platform called ProjectPilot. The platform should combine the best features of Jira, Trello, ClickUp, Notion, and Asana into a single modern web application.

The application must be scalable, secure, responsive, and suitable for large organizations.

TECH STACK
Frontend: Next.js 15, React, TypeScript, Tailwind CSS, ShadCN UI, Zustand, TanStack Query, Framer Motion
Backend: Node.js, Express.js, TypeScript
Database: PostgreSQL, Prisma ORM
Authentication: JWT, Google OAuth
Realtime: Socket.io
Caching: Redis
Storage: Cloudinary
AI: OpenAI API
Deployment: Docker, Docker Compose
